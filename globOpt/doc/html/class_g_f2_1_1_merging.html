Greedily assigns points with GID-s that are not in prims to prims that explain them. Unambiguous assignments go through first, than based on proximity, capped by scale.

Template Parameters

_PointPrimitiveTWraps a point, exposing pos() and dir() functions. Concept: GF2::PointPrimitive.
_PrimitiveTWraps a primitive, exposing pos() and dir() functions. Concept: GF2::LinePrimitive2.
_PointContainerTHolds the points. Concept: std::vector< GF2::PointPrimitive >.
_PrimitiveContainerTHolds the primitives grouped by GID in a two level structure. Concept: std::map< int, std::vector< GF2::LinePrimitive2 > >.
_ScalarFloating point precision of primitives, points, etc. Concept: GF2::PointPrimitive::Scalar.
Parameters
[in,out]pointsContains the points, some assigned, some to be assigned to the primitives in prims.
[in]primsContains some primitives tagged with GID and DIR_GID. GID defines the assignment between points and primitives.
[in]scaleDistance threshold parameter.
[in]mode1: re-assign un-ambiguous points (1 adopter); 2: first re-assign unambiguous, then closest, if inside explaining primitive's scale.
