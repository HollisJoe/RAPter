Detailed Description

template<int _Dim, typename _Scalar = float>
class am::Primitive< _Dim, _Scalar >

Primitive wrapper base class. Requires implementation of pos() and dir() functions. It's also good to implement constructor from default input (i.e. from point and normal for planes)

Template Parameters
_DimLength of internal vector storage. Should be able to fit any primitive.
_ScalarInternal data type. Concept: float.
