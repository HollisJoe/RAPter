Detailed Description

Class to wrap a line with. Implements pos() and dir() functions, and a constructor taking a position and a direction.

Stores 3D position at the first three coeffs, and 3D direction at the second three.

Warning
Note, stores direction, NOT normal.

Calculates the length of the line based on the points in cloud, masked by indices and the distance from point to line threshold.

The method calculates the inliers and selects the most far away point from pos() in both directions.

Template Parameters
_PointTType of point stored in cloud. Concept: "PointPrimitve".
_ScalarPrecision of data in minMax.
_PointContainerPtrTType to store the points to select inliers from. Concept: "std::vector<PointPrimitive>*" .
Parameters
[out]minMaxOutput container holding the two endpoints.
[in]cloudPoint container to look for inlier points in.
[in]thresholdA point in cloud is an inlier, if it is closer than this threshold. Usually the "scale".
[in]indicesOptional input to specify subset of points by indices.
Returns
EXIT_SUCCESS
