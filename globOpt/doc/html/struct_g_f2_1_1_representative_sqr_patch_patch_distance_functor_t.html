Detailed Description

template<typename _Scalar, class _SpatialPatchPatchDistanceFunctorT>
struct GF2::RepresentativeSqrPatchPatchDistanceFunctorT< _Scalar, _SpatialPatchPatchDistanceFunctorT >

Distance is combined by spatial and angular terms according to formula: \( patch\_spatial\_weight^2 \cdot \frac{spat\_dist^2}{spat\_thresh} + \frac{ang\_diff^2}{ang\_thresh} < 1 \). The weight was mostly set to 0.5.
